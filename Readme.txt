
JOE & MAC DEMO (Apr-2000)
-------------------------

1. SUMMARY 
	This is a small (1 level) test of a Joe & Mac kind of clone for the PC
	made with DirectX 6.1. 
	I burned the source code in a cheap CDRW and I lost it forever.

2. REQUIREMENTS
	-DirectX 6.1+
	-Sound Card
	
3. HOW TO PLAY
	-Arrows => movement (up,down,left,right)
	-Ctrl 	=> fire 
	-Space 	=> jump 
	-Esc 	=> exits the game
